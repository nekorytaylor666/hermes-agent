# Scene Type: Dialogue (CS3.5)

For scenes where spoken dialogue drives the drama: confrontations, negotiations, interrogations, confessions, manipulations, comedy exchanges.

---

## Dialogue Archetypes (6 Types)

### Confrontation (shifting power dynamic, escalation)
| Aspect | Rule |
|--------|------|
| Power Dynamic | Asymmetric, shifts across exchange |
| Camera Signature | OTS ping-pong, tight on power holder, axis cross at shift moment |
| Edit Pattern | Quick cuts (1–2s per shot), tight on speaker, widen on challenge, ECU on power-shift reaction |
| Shot Pattern | Funnel: WS → OTS → ECU payoff |
| Compression | Keep power-shift line (UNTOUCHABLE), compress setup to 1 line, reaction 1 line |

### Interrogation (asymmetric control)
| Aspect | Rule |
|--------|------|
| Power Dynamic | Questioner dominant, subject vulnerable |
| Camera Signature | Low-angle on questioner (authority), high-angle on subject (submission), slow tightening |
| Edit Pattern | Slow cuts (2–3s per shot), wide on question, tight on squirm, long holds on answer |
| Shot Pattern | Funnel: WS → OTS (dirty) → ECU (subject sweating) |
| Compression | Keep leading question + one answer, compress follow-up to subtext (nod, avoid eye) |

### Manipulation (hidden control, psychological misdirection)
| Aspect | Rule |
|--------|------|
| Power Dynamic | Manipulator appears passive; victim believes they lead |
| Camera Signature | Dirty single on manipulator (warm, close), clean single on target (exposed) |
| Edit Pattern | Slow dissolves or J-cuts, holds on victim's dawning realization, cuts on emotional beat not dialogue beat |
| Shot Pattern | Tightening spiral: each reverse-angle tighter, no wide reset |
| Compression | Keep seduction/lie lines (UNTOUCHABLE), cut defensive responses to 1 word |

### Confession (surrendering control, emotional release)
| Aspect | Rule |
|--------|------|
| Power Dynamic | Character yields, drops mask |
| Camera Signature | Single close-up tightens as confession deepens, minimal cuts, long takes |
| Edit Pattern | Slow tightening, long holds on face, listener reaction visible in same frame or brief cut |
| Shot Pattern | Reverse funnel: ECU → pull back to two-shot at end (exhale, relief) |
| Compression | Keep confession core (UNTOUCHABLE), cut preamble, listener nods without dialogue |

### Negotiation (balanced, mutual respect)
| Aspect | Rule |
|--------|------|
| Power Dynamic | Symmetrical, both parties have leverage |
| Camera Signature | Symmetrical framing, even ping-pong, slight dolly in on agreement/disagreement |
| Edit Pattern | Medium cuts (2–3s per shot), balanced screen time, cuts on "no"/"yes" or proposal |
| Shot Pattern | Flat hold: two-shot throughout, or symmetrical OTS ping-pong |
| Compression | Keep deal terms (UNTOUCHABLE), cut negotiation padding, condense objections |

### Comedy Exchange (timing-driven)
| Aspect | Rule |
|--------|------|
| Power Dynamic | Irrelevant; timing = everything |
| Camera Signature | Wide two-shot (see both faces), quick cut to close reaction on punchline |
| Edit Pattern | Long holds (4–5s) on setup, sharp cut on punchline (1–2s), fast cut to reaction (0.5–1s) |
| Shot Pattern | Flat hold: two-shot for setup, ECU punchline, ECU reaction, back to two-shot |
| Compression | Setup + punchline UNTOUCHABLE, compress callback to subtext (eye-roll, shrug) |

## LOCKED Camera Style Override

When Camera Style axis is LOCKED (preset or custom), it overrides archetype camera behavior entirely. Archetype still defines power dynamics and shot pattern logic, but camera movement, angle vocabulary, and rig type come ONLY from the LOCKED Camera Style.

Examples:
- Confrontation + Classic Static = power dynamics expressed through blocking and performance within a static master, not OTS ping-pong
- Interrogation + Dreamy Flow = push-in on silence becomes gentle handheld sway
- Comedy + Silent Machine = setup/punchline timing preserved, but movement is imperceptible dolly, not handheld

---

## Chaining 2 Archetypes

For complex scenes where emotional temperature shifts mid-scene:
1. Identify the transition moment (where emotion pivots)
2. Scenes 1–2: Archetype A (e.g. Confrontation)
3. Transition: power-shift culminates, emotional pivot
4. Scenes 3–4: Archetype B (e.g. Confession)
5. Keep power-shift line + confession trigger, compress mid-transition

## Shot Patterns

| Pattern | Structure | Best For |
|---------|----------|----------|
| **Funnel** (default) | WS → OTS ping-pong → ECU payoff | Confrontation, Interrogation |
| **Reverse Funnel** | ECU → pull back to two-shot at end | Confession |
| **Flat Hold** | Two-shot held, cut only for reaction CU | Comedy, Negotiation |
| **Tightening Spiral** | Each reverse tighter, no wide reset | Manipulation, Horror/Noir |

### Funnel Detail
- Scene 1 (0–4s): Wide shot, both characters visible, full environment
- Scene 2 (4–8s): OTS on A (tight, dirty single), B reacts (clean single)
- Scene 3 (8–12s): Ping-pong tightening or ECU sequence on power-shift reaction
- Edit density accelerates: WS = 3–4s hold; OTS = 1.5–2s each; ECU = 0.5–1s each

### Reverse Funnel Detail
- Scene 1: ECU face, interior emotional state visible
- Scene 2: Pull back to mid-shot as character releases emotion
- Scene 3: Two-shot or wider, spatial equilibrium restored
- Slow cuts (3–5s per shot), minimal (3 shots max), long holds

## Dialogue Compression

### Power-Shift Line Identification
Lines where character agency shifts: demand, refusal, confession, revelation, ultimatum. Mark as **UNTOUCHABLE** — never rewrite, soften, translate, or delete.

### Compression Tactics (priority order)
1. Keep power-shift line + one setup line + one reaction line; compress or remove all others
2. Convert dialogue to visual subtext: "I don't believe you" → shake head, step back, cross arms
3. Cut defensive/rationalizing lines first (lowest dramatic weight)
4. Cut repetitive info (if setup explained once, don't repeat on response)
5. Condense multi-clause lines to single-word reactions: "No, absolutely not, never" → "Never."
6. Multi-character scenes (3+ chars): limit each to 2 speaking lines, others silent/reactive

### Word Budget Example
12s Confrontation, 4 exchanges: Budget = 25 words total
- Line 1 (accusation, setup): 6 words (UNTOUCHABLE)
- Line 2 (denial, reaction): 4 words
- Line 3 (pressure, power-shift): 8 words (UNTOUCHABLE)
- Line 4 (admission, payoff): 7 words
- Total: 25 words, 4 exchanges, 3 scenes

## Emotional Inventory (internal planning, never in prompt)

For each character, identify:
- **Want**: what they're trying to get
- **Mask**: how they present (calm? friendly? cold?)
- **Tell**: involuntary physical manifestation (micro-expression, breathing, posture shift)

Map to physical descriptions: Mask = posture, Want = eye direction, Tell = jaw tension, breathing rhythm, hand tremor, shoulder hunch. These manifest as physics ONLY — never emotion labels.

---

## Dialogue Integration in Dynamic Description

Dialogue is woven directly into shot descriptions with voice descriptors — never separated into a standalone section.

Format: `[physical action] + [voice descriptor]: '[dialogue line]' — [reaction]`

Example: `A leans forward, jaw tight, voice low and controlled: 'The money never reached the account.' — B steps back, breath catching, hands raised...`

Rules:
- Voice descriptor before quoted line: "voice low:", "barely audible:", "sharp, cutting:"
- Physical action wraps around dialogue: action before line, reaction after
- User-provided dialogue is **UNTOUCHABLE** — never rewrite, soften, or translate
- Preserve original language of dialogue lines verbatim

## Single-Shot Mode

Triggered by: `shot_count = 1`, "one-take"/"single shot"/"no cuts"/"oner" in prompt, or One Take camera preset.

**Default framing:** Two-shot, both characters visible. Camera 8–12 feet, off-center (rule of thirds).

**Movement options (select one):**
- Slow orbit: 15–30° rotation over duration
- Subtle dolly: 2–3 feet in on emotional peaks, retreats on resolution
- Locked static: no movement; performance and props drive scene

**Rack focus as edit substitute:**
- Shallow DOF: focus pulls from speaker to listener on dialogue handoff
- Timeline: 1–2s on speaker, 0.5s rack to listener, 1–2s on listener response
- Simulates traditional J-cut/L-cut editing without cutting

**Prop/gesture replaces compressed dialogue:** character reaches for glass instead of saying "I'm nervous."

Parallel cap: ≤ 4 simultaneous actions. No scene labels — use temporal markers `(0–2s)`, `(2–4s)`. Density: ≤ 3 sentences per temporal block.

---

## Framing Rules

- **Dirty singles** (OTS visible) = warmth, proximity, shared space → primary speaker or power holder
- **Clean singles** (alone in frame) = vulnerability, exposure → secondary speaker or maximum vulnerability
- **180° rule:** A = frame-left, B = frame-right. All shots respect axis. **Exception:** axis cross at power-shift moment (Confrontation only), justified by character movement
- **Props as subtext:** Establish in Static Description, deploy in Dynamic as gestural beat. Don't over-explain — let audience infer

## Genre Modifiers

| Genre | Tempo & Cuts | Camera | Palette & Light | Sound |
|-------|-------------|--------|-----------------|-------|
| Action | Fast 1–1.5s, sharp | Wide establishing, dynamic, low angles | High contrast, saturated, deep shadows | Impact foley, minimal dialogue |
| Horror | Variable (long holds + sudden cuts) | Dutch angles, ECU, negative space | Desaturated, cold, isolated practicals | Ambient dread, silence as weapon |
| Comedy | Timing-locked (setup 3–4s, punch 1–2s, reaction 0.5–1s) | Wide two-shot, close reaction | Bright, saturated, even exposure | Dialogue crisp, reaction foley |
| Noir | Medium 2–3s, holds on reaction | High contrast, shadows half-face, soft backlight | Desaturated, warm shadows cool highlights | Minimal dialogue, ambient presence |
| Epic | Long 3–5s, dissolves | Wide symmetrical, crane implied | Rich saturated, volumetric light | Grand underscore, spatial dialogue |
| Drama | Medium 2–3s, cuts on emotional beat | Natural light, depth via focus, intimate | Naturalistic palette, mood correction | Dialogue centered, silence as emphasis |

## Overflow Compression (if output > 4000 chars)

**Tier 1 — UNTOUCHABLE (never compress):**
- Starting Composition (mandatory, complete)
- Camera Settings block (as-is)
- User-provided dialogue lines (verbatim, full)
- Power-shift lines in Dynamic Description
- Negative block (all negations present)

**Tier 2 — KEEP IF SPACE (compress, preserve intent):**
- Style & Mood → reduce to 1 sentence
- Narrative Summary → tighten language
- Prop subtext → gesture only, cut explanation
- Reaction descriptions → 1 sentence

**Tier 3 — TRIM FIRST (compress aggressively):**
- Static Description → 1 sentence per zone
- Supporting character descriptions → functional label only
- Transitions → "→" only
- Sound design notes → cut entirely

Rules: merge adjacent scenes, never delete. Convert non-power-shift dialogue to gesture. Never exceed 4000 chars.
