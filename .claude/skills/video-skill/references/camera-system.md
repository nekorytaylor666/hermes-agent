# Seedance 2.0 — Camera System

## Core Principle

Camera movement directly determines video quality. Good camera movement serves story and emotion — never technique for its own sake. In scenes without dialogue or action, **camera IS the narrator.**

---

## Camera Language

### Angles
| Term | Use |
|------|-----|
| low-angle | Power, dominance |
| high-angle | Vulnerability, exposure |
| dutch angle | Unease, moral compromise |
| bird's-eye | Scale, overview |
| worm's-eye | Extreme power, threat |
| eye-level | Neutral, documentary |
| OTS (over-the-shoulder) | Dialogue, spatial relationship |

### Focal Length
| Term | Range |
|------|-------|
| wide | 14–24mm |
| standard | 35–50mm |
| telephoto | 85–200mm |
| macro | Extreme close-up, detail |

### Movement
| Term | Description |
|------|-------------|
| tracking | Camera follows subject laterally |
| dolly-in / push-in | Camera moves toward subject |
| dolly-out / pull-back | Camera moves away from subject |
| crane | Vertical movement up/down |
| pan | Horizontal rotation on axis |
| tilt | Vertical rotation on axis |
| whip-pan | Fast horizontal snap |
| orbit | Camera circles subject |
| handheld | Organic, unsteadied movement |
| Steadicam | Smooth operator-carried movement |
| aerial | Drone/helicopter overhead |
| truck (lateral) | Camera slides sideways on track |

### Time
| Term | Effect |
|------|--------|
| slow-motion | Stretch time at peak moment |
| speed ramp | Variable speed within shot |
| freeze frame | Full stop on single frame |
| timelapse | Compress long duration |
| hyperlapse | Moving timelapse |
| reverse | Playback in reverse |

### Transitions
| Term | Use |
|------|-----|
| smash cut / hard cut | Abrupt direct cut |
| match cut | Visual or motion continuity across cut |
| whip-pan transition | Fast pan bridging two shots |
| L-cut | Audio leads or trails the visual cut |
| dissolve | Gradual blend between shots |

## Cut Rules

### 1. Double Contrast (mandatory on every cut)

Every cut changes **BOTH** shot size **AND** camera character.

**Shot-size scale:** extreme wide → wide → medium → medium close-up → close-up → ECU

**Camera modes:** Handheld | Static/locked-off | Stabilized tracking | Crane/vertical | Aerial/drone — **never repeat across a cut.**

### 2. Re-anchoring and 180° Rule

After cuts returning to established space:
- Re-state who is where, which direction they face
- **180° rule:** character moves left-to-right before cut → same direction after
- State movement direction explicitly after every cut

### 3. Insert Rules

Inserts = sub-second (0.3–0.5s) dramatic punctuation. Any shot size (ECU, extreme wide, contrast angle).

**Rules:**
- Inserts must NOT contain story beats — static moments only
- **Causally motivated:** viewer must understand WHY they see this detail
- **Name the subject:** specify WHOSE body part/detail (Seedance renders wrong character without attribution)
- Obey double contrast
- **Never use** "boot in puddle" — visual cliché

## Speed Ramp Integration

Speed ramps go into Dynamic Description as temporal directives — **NEVER as metadata tags.**

| Directive | Effect | Best For |
|-----------|--------|----------|
| Slow motion | Stretch peak moment | Decisive hit, revelation, detail |
| Speed up / timelapse | Compress time | Travel, transformation, setup |
| Fast to slowmo (decel) | Rush → linger | Approach → impact, travel → arrival |
| Slowmo to fast (accel) | Tension → explosion | Standoff → first move, calm → break |
| Impact speed ramp | Normal → slow at contact → snap back | Collision, physical turning point |
| Freeze to motion | Full freeze → resume | Mid-air, weapon drawn, eye contact |

**Examples in prose:**
- "He charges forward — time stretches as his fist connects, impact rippling in slow motion — then snaps back to real-time as the body hits the ground."
- "The frame freezes on the fist an inch from the jaw — held for a breath — then motion erupts."
- "Timelapse: clouds race across the canyon, shadows sweeping the red rock walls."

## Camera Serves Dramaturgy

Every camera choice needs a narrative reason:
- **Low-angle** = dominance, power
- **High-angle** = vulnerability, exposure
- **Push-in** = increasing tension, intimacy
- **Pull-back** = revelation, isolation
- **Whip-pan** = surprise transfer
- **Dutch angle** = unease, moral compromise
- **Static hold** = deliberate tension, comedy timing
- **Handheld** = urgency, documentary authenticity
- **Steadicam/tracking** = immersion, journey
- **Crane/aerial** = scale, ceremony, epic

## Depth Layering

Every wide shot should have at least 3 depth layers:
1. **Foreground** — frame element, texture, blur
2. **Midground** — subject
3. **Background** — context, environment
