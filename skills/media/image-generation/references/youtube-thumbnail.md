# YouTube Thumbnail

Style reference for generating YouTube video thumbnail images — click-worthy, high-contrast, emotion-driven.

## When to Use

- "youtube thumbnail", "thumbnail", "video preview", "video cover"
- Any request for a video preview image optimized for click-through

## Visual Direction

- **Face:** Exaggerated facial expression (surprise, shock, excitement, curiosity) — fills 30-40% of frame. Eyes wide, mouth open or smiling big
- **Text:** 3-5 words MAX. Huge bold font, stroke/outline for readability. Yellow, white, or red text on contrasting background
- **Color:** Hyper-saturated, warm tones. Red/yellow/orange dominate. Strong color contrast between elements
- **Composition:** Face on one side, text/object on the other. Rule of thirds. No empty space — every pixel earns its place
- **Background:** Blurred, gradient, or solid color. Never busy. Subject pops against it
- **Arrows/circles:** Optional — pointing at key element, red circle highlighting something

## Prompt Structure

```
[Person with exaggerated expression], [bold text overlay: "TEXT HERE"],
[bright saturated colors, yellow/red accents], [clean background, slight blur],
[YouTube thumbnail style, high contrast, eye-catching, clickbait aesthetic],
[aspect ratio 16:9]
```

## Key Rules

- Always 16:9 aspect ratio (1280x720 standard YouTube thumbnail)
- Maximum 3-5 words of text — if more, it's unreadable at thumbnail size
- Face + emotion = clicks. Always include a face if possible
- High saturation — thumbnails compete with hundreds of others, need to pop
- Test mental model: "Would I click this at 120px wide in a sidebar?"
- Never use small text, thin fonts, or low contrast
