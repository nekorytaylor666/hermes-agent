# Poster Design

Style reference for generating promotional poster images — event posters, movie posters, product launch posters, social media banners.

## When to Use

- "make a poster", "poster design", "promotional image", "event banner", "launch poster"
- Any request for a static visual with text, branding, and graphic layout

## Visual Direction

- **Composition:** Bold, eye-catching, clear visual hierarchy. Hero element (product/person/scene) takes 50-60% of frame
- **Typography:** Large, readable headline. Limit to 2-3 text elements (headline, subline, CTA/date). High contrast text against background
- **Color:** Strong brand colors or thematic palette. High saturation, bold contrasts
- **Layout:** Clean negative space around text. No clutter. Every element serves a purpose
- **Style:** Professional graphic design aesthetic — not a photo, not a painting

## Prompt Structure

```
[Subject/hero element description], [layout description], [typography: headline text, subline],
[color palette], [style: professional poster design, clean layout, bold typography],
[technical: high resolution, print-ready quality]
```

## Key Rules

- Always include actual text content the user wants on the poster
- Specify text placement (top, bottom, overlay, sidebar)
- Use high contrast between text and background for readability
- Keep text short — posters are visual-first, text-second
- Aspect ratio: typically `3:4` (portrait poster) or `16:9` (banner)
